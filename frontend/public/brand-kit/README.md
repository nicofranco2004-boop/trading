# Rendi · Brand Kit

Sistema de identidad de Rendi · v1.0 · Mayo 2026

## Contenido

- `logos/` — Isotipo en 4 variantes PNG (violeta, blanco, ink, original)
- `tokens.css` — Variables CSS listas para copiar/pegar
- `tokens.json` — Tokens en JSON (compatibles con Figma, Tailwind, Style Dictionary)
- `fonts.md` — Instrucciones para instalar Geist + JetBrains Mono
- `manual.html` — Manual completo navegable

## Cómo usar

1. **Logo** — Usá la variante que mejor contraste con tu fondo:
   - Fondo oscuro (ink) → `rendi-mark-violet.png`
   - Fondo violeta → `rendi-mark-ink.png`
   - Fondo claro → `rendi-mark-ink.png`
   - Para impresión → `rendi-mark-original.png`

2. **Colores** — Importá `tokens.css` o copiá los valores que necesites.

3. **Tipografía** — Seguí `fonts.md` para instalar Geist y JetBrains Mono.
